## Harmony Heights Band Webpage

### Project Overview
Official webpage for Harmony Heights, an emerging band known for their eclectic mix of genres and compelling live performances. The webpage serves as a central hub for fans to explore music, learn about band members, view tour dates, and connect with the band.

### Features
- **Band Introduction**: Welcome section with captivating description
- **Photo Gallery**: Professional photos of the band and individual members
  - Full band photo
  - Amy Duncan (Vocals/Sampling)
  - John Smith (Bass/Guitars)
  - Trevor G. (Drums)
- **Audio Player**: Stream latest tracks directly from the website
- **Video Player**: Watch live performances and music videos
- **Tour Dates**: Upcoming shows with ticket purchase links
- **Social Media Links**: Connect on Faceblock, Instagrump, and Twotter
- **Contact Form**: Send messages or booking inquiries

### Technical Specifications
- **Language**: HTML5
- **Styling**: CSS3 (embedded)
- **Responsive Design**: Mobile-friendly grid layout
- **Accessibility**: Alt text for all images, semantic HTML
- **File Name**: `harmony-heights.html`

### Design Elements
- **Color Scheme**: 
  - Purple gradient background (#667eea to #764ba2)
  - Gold accents (#ffd700) for highlights
  - White text for readability
- **Typography**: Arial, sans-serif
- **Effects**: 
  - Hover animations on images
  - Smooth transitions
  - Semi-transparent containers
  - Text shadows for depth

### File Structure
```
harmony-heights.html
├── Head Section
│   ├── Meta tags (charset, description)
│   ├── Title
│   └── Embedded CSS
└── Body Section
    ├── Band Introduction
    ├── Photo Gallery
    ├── Audio Player
    ├── Video Player
    ├── Tour Dates
    ├── Social Media Links
    └── Contact Form
```

### Form Fields
1. **Name**: Text input
2. **Email**: Email input with validation
3. **Subject**: Text input
4. **Message**: Textarea for longer messages
5. **Submit Button**: Sends form data

### External Resources
- Band photos hosted on edube.org
- Audio track (MP3 format)
- Performance video (MP4 format)
- Ticket purchasing links to example.com

### Browser Compatibility
- Supports all modern browsers
- Fallback messages for unsupported audio/video elements
- HTML5 required for multimedia features

### Usage Instructions
1. Open `harmony-heights.html` in any web browser
2. Navigate through sections using scroll
3. Click "Get Tickets" links to purchase tickets (opens in new tab)
4. Use audio/video controls to play media
5. Fill out contact form to send messages
6. Click social media links to follow the band

### Customization Options
- Modify colors in the CSS `<style>` section
- Update tour dates in the unordered list
- Replace image URLs with new photos
- Add more audio/video tracks
- Adjust grid layout for different screen sizes
